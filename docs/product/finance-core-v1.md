# Finance Core v1

## Objetivo

Entregar uma plataforma web protegida por login para visualizar finanças pessoais e empresariais separadamente.

## Inclui

- Login por e-mail/senha.
- Dashboard pessoal.
- Dashboard empresarial.
- Lista de movimentações com filtros.
- Revisão de lançamentos sem categoria.
- Configuração de contas, categorias e integrações.
- Importação CSV para histórico.
- Sincronização Banco MCP para Open Finance.
- Registro de importações e sincronizações.

## Não inclui

- Facebook Ads.
- Tracking tipo Utmify.
- Gestão de criativos.
- Gestão de ofertas.
- Checkout ou conciliação por pedido.
- Multiusuário ou multiempresa.

## Critério de sucesso

Kelvenyn deve conseguir entrar na plataforma, escolher pessoal ou empresa, ver entradas, saídas, saldo, categorias e revisar lançamentos pendentes sem misturar contextos.
